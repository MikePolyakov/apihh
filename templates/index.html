HeadHunter API

С помощью HeadHunter API вы можете получать информацию о соискателях и компаниях,
а также использовать функциональность HeadHunter для своего сайта или приложения.

Общая информация

  • Всё API работает по протоколу HTTPS.
  • Все данные доступны только в формате JSON.
  • Базовый URL — https://api.hh.ru/
  • Даты форматируются в соответствии с ISO 8601.
